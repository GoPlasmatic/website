Create a high-fidelity landing page design in Framer for a product called Orion, using an existing premium design system.

## Positioning
Orion is:
“The nervous system for modern software.”

It is a runtime layer that coordinates and governs system behaviour.

---

## Design Intent
This is NOT a wireframe.

Design a polished, production-ready page:
- refined typography
- strong visual hierarchy
- intentional spacing
- premium motion
- minimal but expressive UI

Style:
- dark, cinematic, high-contrast
- subtle glow and signal-based visuals
- abstract system aesthetic (no literal anatomy)
- clean, modern, design-forward

---

## Core Experience Narrative
The page should visually and conceptually move through:

uncoordinated signals → emergence of control → fully coordinated system

Everything must reinforce:
Orion enables systems to behave as one
